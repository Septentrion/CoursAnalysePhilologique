# Vision par ordinateur appliquée à l'analyse textuelle

Ce dépôt contient des ressources pour un cours sur la vision par ordinateur.
Le terrain d'expérimentation choisi estcelui de la lecture de textes numérisés,
et particulièrement, ici, de manuscrits anciens.

Ce cours s'inscrit dans une séquence plus longue, composée de deux versants.
Outre la vision par ordinateur, les résultats obtenus avec les techniques développées ici
seront utilisés dans un cours sur le traitement des langues naturelles.
Logiquement, l'application de ce second cours sera d'analyser et de comprendre les textes transcrits
par le système de reconnaissance et de segmentation d'images.

Les supports de ce dépôt sont organisés en cinq chapitres :

1. Quels sont les problèmes spécifiques de la philologie ? Comment la machine peut-elle les résoudre ?
2. Qu'est-ce que la vision par ordinateur ? Quelles sont les architectures logicielles disponibles ?
3. Revue de quelques outils courants utilisés pour la vision par ordinteur
4. Comment construire un processus de traitement des textes numérisés ?
5. Éléments de génie logiciel relatifs à cetype d'application
